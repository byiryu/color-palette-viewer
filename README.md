# weather-outfit-color-tool

`weather-outfit` 앱 V4-1 색상 알고리즘 검증용 시각화 도구.

날씨 기반 코디 추천 앱의 색상 시드 데이터(약 105색)를 시각적으로 검토할 수 있는 정적 HTML 페이지입니다.

## 사용

GitHub Pages: https://byiryu.github.io/weather-outfit-color-tool/

## 기능

- 색상 카드 그리드 (스와치 + HEX + 한국어/영어 이름)
- 필터: 날씨 / 기온 / 계절 / 역할
- 태그: warmth, tone, role, weather_fit, temp_fit, season_fit, scheme_fit

## 데이터 출처

- Pantone Color of the Year 2010-2026 (검증된 HEX)
- Pantone Fashion Color Trend Report SS26 (TCX → HEX 근사)
- Claude 학습 데이터 기반 패션 베이직 컬러 (developer seed)
