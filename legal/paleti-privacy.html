PALETI 개인정보 처리방침

서비스명: PALETI (한국 모바일 앱 — 색으로 시작하는 하루)
운영자: byiryu studio · 연락처 byiryustudio@gmail.com
시행일: 2026-05-18
최종 갱신: 2026-05-18 (v1.0)

byiryu studio (이하 "회사")는 PALETI 모바일 앱 (이하 "서비스") 이용자(이하 "사용자")의 개인정보를 매우 중요하게 다루며, 개인정보보호법 및 관련 법령을 준수합니다. 본 처리방침은 회사가 사용자의 개인정보를 어떤 목적으로 수집·이용하며, 어떻게 처리하고 어떤 권리를 보장하는지 안내합니다.

1. 처리 목적

회사는 다음 목적으로 개인정보를 처리합니다.

목적 사용 정보
회원 가입 및 계정 관리 Google 계정 식별자 (sub), 가입 이메일
맞춤형 색상·옷차림 추천 사용자가 입력한 성별, GPS 위치 좌표, 컬러 테마 선호
위치 기반 날씨·미세먼지 정보 제공 GPS 좌표 → 기상청 격자 / 에어코리아 측정소 매칭
푸시 알림 발송 FCM 등록 토큰, 알림 설정 (모닝푸시·미세먼지 경고)
광고 노출 및 매출 광고 식별자 (Android AAID / iOS IDFA, ATT 동의 시)
서비스 품질 향상 익명 크래시 정보, 추천 이력 (개인 식별 불가 형태)

3. 개인정보 보유 및 이용 기간

  • 계정 정보 (Google sub, email, 성별, 위치 등): 회원 탈퇴 시점에 즉시 삭제 (auth.users ON DELETE CASCADE 로 모든 연관 테이블 자동 삭제)
  • 추천 캐시: 1시간 후 자동 만료
  • 날씨 캐시: 1시간 후 자동 만료
  • 추천 이력 (개인 식별 가능 형태): v1.5 익명화 후 30~90일 자동 삭제 (정확한 기간은 v1.5 도입 시점에 결정)
  • 법령상 보존 의무 정보: 해당 시 (현재 v1.0 IAP 결제 없음 — 보존 의무 0)

4. 제3자 제공 및 처리 위탁

회사는 사용자의 동의 없이 제3자에게 개인정보를 제공하지 않습니다. 단, 서비스 제공에 필수적인 외부 시스템에 다음 정보를 위탁·송신합니다.

API 송신 정보 라이선스 면책

5. 위치정보 처리 (위치정보보호법 정합)

본 서비스는 사용자의 GPS 좌표를 수집·이용하므로 위치정보보호법의 적용을 받습니다.

항목 내용
수집 방식 사용자의 명시적 동의 후 (Android ACCESS_FINE/COARSE_LOCATION / iOS NSLocationWhenInUseUsageDescription) 포그라운드만 (백그라운드 위치 수집 X)
수집 시점 메인 화면 진입 시 1회 (사용자가 새로고침 또는 자동 갱신을 명시적으로 요청한 경우에만 추가 수집)
수집 목적 현재 위치 날씨·미세먼지 정보 제공 및 색상 추천
이용 범위 격자 단위 (약 5km × 5km) 익명화 후 캐시 — 정확한 좌표는 추천 응답 후 즉시 폐기
위치 정보 제공자 (수탁사) server 측 (Supabase) 만 — 외부 송신은 격자 단위 익명화 좌표만 (4.2 항목)
위치 정보 수집 거부 사용자는 언제든지 시스템 설정 또는 앱 마이페이지에서 위치 권한을 철회할 수 있습니다. 거부 시 "도시 직접 선택" 모드로 서비스 이용 가능 (자동 GPS 없음)
GPS 변동 자동 갱신 5km 이상 위치 변동 시 자동 새 좌표 fetch (앱 진입 또는 새로고침 시) — 명시적 사용자 행위 외 백그라운드 수집 없음

6. 자동화된 의사결정 (AI 추천)

본 서비스는 AI (Anthropic Claude Haiku 4.5) 기반 자동화된 의사결정으로 사용자에게 색상·옷차림을 추천합니다.

항목 내용
결정 대상 오늘의 4가지 옷 색상 (외투·상의·하의·신발) + 옷 종류 텍스트 추천
결정 근거 사용자가 입력한 성별, 컬러 테마, 현재 GPS 위치의 날씨·미세먼지·시간대
사용자에게 미치는 영향 옷차림 결정 보조 — 사용자가 추천을 따를지 여부를 자유롭게 결정. 본 서비스가 사용자의 실 의류 구매·소비 결정을 강제하지 않음
이의 제기 권리 사용자는 추천 결과에 대해 새로고침을 통해 재추천을 요청하거나, 마이페이지에서 컬러 테마를 변경할 수 있습니다. 자동화된 결정에 대한 설명·이의 제기를 byiryustudio@gmail.com 으로 요청할 수 있습니다.

7. 권한 사용 안내

본 서비스는 다음 시스템 권한을 사용하며, 각 권한의 사유와 거부 시 영향을 안내합니다.

권한 사용 사유 거부 시 영향
위치 (포그라운드) 현재 위치 날씨로 색상 추천 자동 GPS 모드 불가 — "도시 직접 선택" 모드로 사용 가능 (수동 도시 선택)
알림 (Android 13+ / iOS) 미세먼지 매우 나쁨 알림 + 모닝푸시 (선택 사용자) 알림 미수신 — 다른 기능 정상
광고 추적 (iOS ATT / Android 광고 ID) 맞춤형 광고 노출 비타겟 광고 표시 (광고 ID 사용 X) — 다른 기능 정상
인터넷 서비스 운영 필수 (날씨·미세먼지·추천 API 호출) 오프라인 시 서비스 미지원 (캐시 만료 후 동작 X)

8. 만 14세 미만 사용자 처리

본 서비스는 만 14세 이상의 사용자를 대상으로 합니다. 위치정보보호법 제25조에 따라 만 14세 미만의 개인정보를 처리하지 않습니다.

  • Google Sign-In 기준 가입 단계에서 만 14세 미만으로 확인되는 경우 가입을 차단합니다 (Google 정책 정합).
  • 만 14세 미만 사용자의 개인정보가 의도치 않게 처리되었음이 확인되는 경우 즉시 삭제 및 향후 처리 중단 조치합니다.

9. 개인정보의 안전성 확보 조치

회사는 다음 조치로 개인정보의 안전성을 확보합니다.

조치 내용
암호화 저장 Supabase 데이터베이스의 자체 암호화 (AES-256, 디스크 레벨)
암호화 전송 클라이언트↔server 모든 통신 TLS 1.2+ (HTTPS)
Row Level Security (RLS) Supabase Postgres 의 RLS 정책으로 사용자별 데이터 격리 (auth.uid() = user_id)
접근 제어 운영자 외 server 데이터베이스 접근 권한 부재 (Supabase Dashboard 인증 + 2FA)
광고 ID 익명화 AdMob 광고 식별자는 외부 (Google) 에서만 처리, server 측 직접 처리 X
자체 식별 정보 미보관 주민등록번호·전화번호·신용카드 등 민감/고유식별정보 수집·보관 X
