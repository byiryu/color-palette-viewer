PALETI 이용약관

서비스명: PALETI (한국 모바일 앱 — 색으로 시작하는 하루)
운영자: byiryu studio · 연락처 byiryustudio@gmail.com
시행일: 2026-05-18
최종 갱신: 2026-05-18 (v1.0)

제1조 (목적)

본 이용약관(이하 "약관")은 byiryu studio (이하 "회사")가 운영하는 모바일 애플리케이션 PALETI (이하 "서비스")의 이용 조건과 절차, 회사와 이용자의 권리·의무 및 책임 사항을 규정함을 목적으로 합니다.

제2조 (정의)

본 약관에서 사용하는 용어의 정의는 다음과 같습니다.

용어 정의
서비스 PALETI 모바일 앱 — GPS 기반 날씨·미세먼지 데이터 + AI 색상·옷차림 추천 서비스
이용자 본 서비스에 접속하여 이 약관에 따라 서비스를 이용하는 자
회원 Google 또는 Apple 계정으로 로그인하여 서비스를 이용하는 자
콘텐츠 본 서비스가 제공하는 색상 추천, 옷 종류 텍스트, 위젯 표시 정보 등 일체의 정보
AI 추천 Anthropic Claude Haiku 4.5 API 를 활용한 자동화된 색상·옷차림 추천 결과

제3조 (약관의 효력 및 변경)

  1. 약관 효력 발생 — 본 약관은 회사가 서비스를 통하여 공개함으로써 효력이 발생합니다. 회원은 회원 가입 시점에 본 약관에 동의한 것으로 간주됩니다.

  2. 약관 변경 — 회사는 관련 법령을 준수하는 범위 내에서 본 약관을 변경할 수 있습니다. 변경 약관은 다음 절차로 고지·시행됩니다.
    - 변경 시행일 7일 전 (사용자에게 불리한 변경의 경우 30일 전) 앱 내 공지 + 가입 이메일 통지
    - 변경 시행 후 회원의 계속 이용 행위는 변경 약관에 대한 동의로 간주
    - 변경 약관에 동의하지 않는 회원은 회원 탈퇴를 통해 서비스 이용을 중단할 수 있음

제4조 (회원 가입 및 계정 관리)

  1. 가입 방식 — 본 서비스는 Google 소셜 로그인 및 Apple 소셜 로그인 (iOS 앱 한정) 만 제공합니다. 이메일·전화번호 직접 가입은 지원하지 않습니다.

  2. 가입 자격 — 본 서비스는 만 14세 이상의 자연인만 가입할 수 있습니다. 만 14세 미만의 사용자는 가입이 차단됩니다 (Google 정책 정합).

  3. 계정 보안 — 회원은 자신의 Google·Apple 계정에 대한 보안 책임을 지며, 본인 계정의 무단 사용으로 인한 손해에 대해 회사는 책임지지 않습니다.

  4. 회원 정보 변경 — 가입 이메일·기본 위치·컬러 테마·알림 설정 등은 마이페이지에서 회원이 직접 변경할 수 있습니다.

제5조 (서비스의 제공)

회사가 제공하는 서비스는 다음과 같습니다.

항목 내용
위치 기반 날씨 정보 기상청 단기예보 API 를 통한 기온·강수·풍속·하늘 상태·습도 정보
미세먼지 정보 한국환경공단 에어코리아 API 를 통한 PM10/PM2.5 농도·등급
AI 색상 추천 Anthropic Claude Haiku 4.5 기반 4가지 옷 색상 (외투·상의·하의·신발) + 4가지 옷 종류 텍스트 추천
컬러 테마 선택 10종 (오리지널·모노크롬·팝·파스텔·어스톤·로맨틱·쿨스트릿·마린·빈티지·포레스트)
위젯 Android Glance / iOS WidgetKit 홈 화면·잠금 화면 위젯 (현재 위치 + 추천 색상 표시)
알림 미세먼지 매우 나쁨 경고 (선택), 모닝푸시 (선택)
18시 이후 다음날 예보 자동 전환 한국 KST 18시 이후 앱 진입 시 내일 09:00 KST 예보 자동 채택

서비스는 무료로 제공되며, 광고 매출 (Google AdMob 배너) 으로 운영됩니다. v1.x 이후 크레딧·구독 등 유료 기능이 도입될 수 있으며, 이 경우 별도 약관 동의를 받습니다.

제6조 (서비스의 변경 및 중단)

  1. 서비스 변경 가능 — 회사는 서비스의 일부 또는 전부를 변경, 추가, 중단할 수 있습니다. 변경 시 사전 고지합니다 (긴급 사유 제외).

  2. 외부 의존 서비스 중단 가능성 — 본 서비스는 다음 외부 시스템에 의존하며, 이들의 서비스 중단·정책 변경 시 본 서비스도 제한·중단될 수 있습니다.
    - 기상청 단기예보 API (공공누리 제1유형) — 정확성·지속성 비보장
    - 한국환경공단 에어코리아 API (공공누리 제1유형) — 정확성·지속성 비보장
    - Anthropic Claude API — 호출 한도·정책 변경 시 영향
    - Supabase Cloud — Free tier 한도 초과 시 일시 중단 가능
    - Google AdMob / Firebase — 정책 변경 시 영향

  3. 장애 발생 시 대응 — 외부 서비스 장애 시 회사는 가능한 범위 내에서 캐시 데이터 또는 폴백 (매트릭스 룰) 으로 서비스를 유지하며, 장애 사실을 앱 내 공지합니다.

제7조 (이용자의 의무)

이용자는 다음 행위를 하여서는 안 됩니다.

  1. 본 서비스에 대한 무단 접근·해킹·역공학 시도
  2. 본 서비스를 자동화 도구·봇 등으로 부정 이용
  3. 본 서비스의 콘텐츠를 무단 복제·배포·상업적 이용
  4. 다른 회원의 계정 도용
  5. 회사의 운영을 방해하는 일체의 행위
  6. 관련 법령·약관·이용 정책에 반하는 행위

위 행위 시 회사는 사전 통보 없이 서비스 이용을 제한·중단할 수 있습니다.

제8조 (회사의 의무)

  1. 회사는 본 약관 및 관련 법령을 준수하며, 사용자에게 안정적·지속적인 서비스를 제공하기 위해 최선을 다합니다.
  2. 회사는 사용자의 개인정보를 보호하며, 별도의 개인정보 처리방침 을 통해 처리 내역을 공개합니다.
  3. 회사는 사용자의 권리 행사 (열람·정정·삭제·처리정지·이의 제기) 에 대해 영업일 기준 10일 이내 회신합니다.

제9조 (책임 제한 및 면책)

  1. 공공 데이터 정확성 면책 — 본 서비스의 날씨·미세먼지 정보는 기상청·한국환경공단의 공공 API 를 활용합니다. 이들 데이터의 정확성·지속성은 제공기관이 보장하지 않으며, 회사는 데이터 오류·지연·중단으로 인한 손해에 책임을 지지 않습니다.

  2. AI 추천 면책 — AI 색상·옷차림 추천은 참고 자료입니다. 회사는 추천 결과의 적합성·정확성을 보증하지 않으며, 사용자는 본인의 판단으로 의류를 결정합니다. 추천 결과에 따른 직접·간접 손해 (예: 외출 시 추위·더위, 사회적 인식) 에 책임을 지지 않습니다.

  3. 외부 서비스 면책 — 본 서비스가 의존하는 외부 시스템 (Supabase, Anthropic, Google, Firebase 등) 의 장애·정책 변경으로 인한 손해에 회사는 책임을 지지 않습니다.

  4. 광고 면책 — 본 서비스의 광고 (Google AdMob) 는 광고주의 책임이며, 회사는 광고 내용·정확성·광고를 통한 거래에 책임을 지지 않습니다.

  5. 무료 서비스 면책 — 본 서비스는 v1.0 기준 무료로 제공되며, 무료 서비스 특성상 회사의 손해배상 책임은 고의·중과실에 의한 경우에 한정됩니다.

  6. 천재지변·불가항력 면책 — 천재지변, 전쟁, 통신 장애, 정부 정책 변경 등 회사의 통제 범위를 벗어난 사유로 인한 서비스 중단·손해에 회사는 책임을 지지 않습니다.

제10조 (광고)

  1. 본 서비스는 메인 화면 하단에 Google AdMob 배너 광고를 노출합니다.

  2. 광고 노출은 회원 자격 유지의 일부이며, 별도 동의 없이 노출됩니다 (단, iOS 의 경우 ATT 다이얼로그 동의 후 맞춤 광고 노출 — 거부 시 비타겟 광고).

  3. v1.x 이후 광고 제거 옵션 (유료 구독 등) 이 도입될 수 있습니다.

제11조 (유료 결제 — v1.x 이후)

현재 v1.0 기준 본 서비스는 무료입니다. v1.x 이후 AI 코디 이미지 생성, 크레딧 충전, 구독 등 유료 기능 도입 시 다음 사항을 적용합니다.

  1. 결제 시스템 — Google Play Billing (Android) / Apple StoreKit (iOS) 만 사용. 외부 결제 시스템 미지원.
  2. 환불 정책 — Google Play / Apple App Store 의 환불 정책을 따릅니다.
  3. 결제 영수증 검증 — server 측에서 영수증 검증 후 크레딧 부여 또는 구독 활성화.
  4. 세부 약관 — 유료 결제 도입 시 본 약관에 별도 조항 추가 + 사전 동의 절차 수행.

제12조 (회원 탈퇴 및 계정 삭제)

  1. 회원 탈퇴 방법 — 마이페이지의 "회원 탈퇴" 메뉴를 통해 즉시 탈퇴할 수 있습니다.

  2. 즉시 삭제 — 회원 탈퇴 시 다음 데이터가 즉시 삭제됩니다 (개인정보 처리방침 정합).
    - Google·Apple 계정 식별자 (sub, email)
    - 성별, 컬러 테마, 알림 설정, 기본 위치
    - FCM 등록 토큰
    - 추천 이력 (CASCADE 삭제)

  3. 삭제 불가 데이터 — 외부 시스템에 위탁된 데이터 (Google AdMob 광고 식별자, Firebase Crashlytics 크래시 정보 등) 은 해당 사업자의 정책에 따라 별도 삭제 요청해야 합니다.

  4. 재가입 — 동일 Google·Apple 계정으로 재가입 가능하나, 이전 데이터는 복원되지 않습니다.

제13조 (손해배상)

  1. 회사의 손해배상 범위 — 회사의 고의 또는 중과실로 인한 사용자의 손해에 대해서만 손해배상 책임을 집니다. 그 범위는 3개월간 본 서비스를 통한 사용자의 직접 결제 금액 합계 (현재 v1.0 무료 = 0원) 를 한도로 합니다.

  2. 사용자의 손해배상 의무 — 사용자가 본 약관 또는 관련 법령을 위반하여 회사 또는 제3자에게 손해를 발생시킨 경우 사용자는 손해배상 책임을 집니다.

제14조 (준거법 및 관할)

  1. 준거법 — 본 약관의 해석 및 본 서비스 이용과 관련된 분쟁에는 대한민국 법이 적용됩니다.

  2. 합의관할 — 본 약관에 관한 소송의 합의관할 법원은 서울중앙지방법원으로 합니다.

  3. 소비자 보호 — 사용자가 소비자인 경우 사용자의 주소지 또는 거소지 관할 법원에 소를 제기할 수 있습니다 (민사소송법 제8조, 소비자기본법 정합).

제15조 (기타)

  1. 본 약관에 명시되지 않은 사항은 byiryu studio 공통 운영 정보 (studio-common.md), 별도 개인정보 처리방침, 관련 법령 및 일반 상관례를 따릅니다.

  2. 본 약관의 일부 조항이 무효 또는 효력을 잃는 경우 나머지 조항은 계속 효력을 가집니다.

  3. 연락처: byiryustudio@gmail.com (영업일 기준 3~5일 이내 응답)
